A Personal Finance Tracker is a responsive web application that helps users manage their daily finances efficiently. The application allows users to add expenses, set monthly budgets, monitor remaining balance, and visualize spending patterns using charts and analytics.

🛠️ Technologies Used

Frontend:

•HTML5: Structure of the web pages

•CSS3: Styling and responsive design

•JavaScript (ES6): Dynamic functionality

🚀 How to Run the Project:

Step 1:

Install: VS Code, Live Server Extension

Step 2:

Open project folder in VS Code.

Step 3:

Click: Open with Live Server
